MacOS常用软件记录

发表于 2023-07-21 更新于 2024-04-09 分类于 开发者手册 阅读次数： 评论数：

💡本文整理记录一下工作中以及日常生活中常用的一些软件和下载链接

操作系统-[镜像]

  • Windows：✨MSDN-itellyou

推荐使用LTSC版本的操作系统，系统比较纯净，但需要激活

  • Linux：CentOS

比较推荐直接去一些软件源上下载，这里举例清华源

操作系统工具-[Mac]

日常使用：

  • 视频会议：✨腾讯会议
  • 文件解压缩：MacZip
  • 待办提醒：苹果提醒事项
  • 电子邮件：苹果邮件
  • 聊天工具：微信、企业微信
  • 翻译：有道词典
  • VPN：✨LetsVPN、EasyConnect
  • 备忘录：苹果备忘录
  • 日程管理：苹果日历
  • 写作工具：✨Typora
  • 图床：iPic
  • 输入法：搜狗输入法
  • 软件包管理：Homebrew

开发工具：

  • 代码编辑器：✨IDEA、✨VSCode
  • Git：Git、Svn
  • 浏览器：Firefox、✨Google Chrome
  • 虚拟化：Docker、✨Parallels Desktop
  • 终端工具：✨FinalShell
  • 数据库工具：DBeaver、RDM、✨Navicat、达梦数据库
  • 远程工具：Microsoft Remote Desktop
  • 数据库设计工具：✨PDManer
  • 接口测试工具：Postman、Jemter

操作系统工具-[Windows]

  • 文件查找搜索：✨Everything

网页工具

  • ✨腾讯杂项在线工具：杂项工具箱
  • ✨画图工具：ProcessOn
  • ✨微信读书

浏览器插件

  • ✨AdGuard AdBlocker拦截广告
  • ✨FireShot 捕捉网页截图
  • ✨GitZip for github 下载GitHub部分文件
  • ✨Wappalyzer 分析网站所使用的组件
