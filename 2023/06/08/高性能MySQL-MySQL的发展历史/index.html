MySQL历史

MySQL 由一家瑞典公司MySQL AB创建，该公司由瑞典人David Axmark、Allan Larsson 和芬兰瑞典人Michael Monty Widenius创立。

Widenius 和 Axmark 最初开发 MySQL 始于 1994 年。

MySQL 的第一个版本出现于 1995 年 5 月 23 日。它最初是基于低级语言ISAM从创建供个人使用，创建者认为它太慢了并且不灵活。他们创建了一个新的SQL接口，同时保留了相同的API作为 mSQL。通过使 API 与 mSQL 系统保持一致，许多开发人员能够使用 MySQL 而不是（专有许可的）mSQL 前身。
